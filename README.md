# Effortless

A macOS menubar app that helps you declare intentions and timebox focus sessions. Full-screen altar overlay transforms your screen into a temple where your attention is given on the altar.

## Install

```
brew tap iulspop/effortless
brew install --cask effortless
```

> **Note:** The app is not code-signed. On first launch, right-click → **Open** or allow it in **System Settings → Privacy & Security**.

## Features

- **Altar** — Full-screen overlay asks "What outcome do you seek?" and locks down Mission Control, Cmd+Tab, and Dock
- **Timebox** — Type minutes or quick-select with Q (5 min), W (25 min), E (90 min)
- **Chalice** — Floating widget shows your intention and countdown timer. Click to cycle screen corners, drag to move freely
- **Menu Bar** — Live countdown timer and session controls
- **Settings** — Dark/Light/System appearance, toggle floating chalice on/off
- **Session Log** — All sessions logged to `~/Library/Application Support/Effortless/`
